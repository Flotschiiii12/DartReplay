# 🎯 DartReplay 🎯

Replay every great moment.

DartReplay ist eine Plattform zur automatischen Analyse von Dartspielen, Erkennung von Highlights und Erstellung professioneller Replay-Videos.

## Features

### 📊 Analytics

- Average
- First 9 Average
- Checkout %
- High Finishes
- Matchhistorie
- Rekorde

### 🎬 Replay Engine

- Automatische Highlight-Erkennung
- Replay-Erstellung
- Zwei-Kamera-System
- Ringpuffer-System
- FFmpeg Video Processing

### 🤖 Integrationen

- Discord Bot
- WhatsApp (geplant)

### 📱 Social Media

- TikTok Ready Exporte
- Thumbnail Erstellung
- Hashtag Generierung

## Geplanter Technologiestack

- Python
- FastAPI
- PostgreSQL
- Docker
- Linux
- FFmpeg
- Discord.py
- React
